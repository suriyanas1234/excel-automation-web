TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: Script says "Repository not found"
A: Make sure you created the GitHub repo first (Step 1) before running DEPLOY.ps1

Q: GitHub Pages not showing up
A: Check that the repo is PUBLIC (not private). Make sure you enabled Pages
   in Settings and waited 1-2 minutes.

Q: Website is blank or shows 404
A: Wait a few more minutes for Pages to deploy. Refresh browser.
   Also check that Pages is enabled for "main" branch, "/" folder.
